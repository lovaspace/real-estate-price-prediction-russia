# real-estate-price-prediction-russia
В рамках проекта проведён анализ рынка жилой недвижимости РФ на основе датасета из 5,5 млн объявлений. Цель — построить модель прогнозирования стоимости с учётом факторов ценообразования, сезонных и региональных закономерностей. Проект охватывает полный аналитический цикл: от предобработки и EDA до статистических тестов и машинного обучения.
